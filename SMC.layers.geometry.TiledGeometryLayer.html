Class: TiledGeometryLayer

SMC.layers.geometry. TiledGeometryLayer

(abstract) new TiledGeometryLayer(options)

Base class for layers using client side rendering of tiles containing geographical features in the SMC map viewer component. The tiles contents will be retrieved using a data provided supporting tiling as needed to cover the viewing area.
